Crossroads Demo

The salon site built for fast creative iteration.

A static GitHub Pages starter for a color bar, salon, and wellness retail experience.

View Workflow Research Rules

What this demo is for

This repo is the website foundation for The Crossroads. It keeps the stack simple so the owner can focus on creative direction while agents handle implementation and sourced research.

Hard guardrails

  • No made-up market data or competitor facts.
  • No health or efficacy claims without credible sources.
  • All factual summaries must cite the web or say none were found.

Owner Workflow

Creative in, build out.

  1. The owner gives direction, offers, visuals, and approvals.
  2. The build agent updates the static site.
  3. The research agent gathers live sourced information.
  4. The strategy agent turns sourced findings into clear next steps.

Static-first

This GitHub Pages version is intentionally light: HTML, CSS, and a small script layer. It is easy to preview, version, and hand off.

Next phase

If needed, this can later be upgraded to a richer stack without discarding the content, prompts, or research policy.
